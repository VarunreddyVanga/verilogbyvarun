VERILOG

Module

  • A module is basic block in verilog.
  • A module can be element, or collection of low level design.
  • Typically elements are grouped into modules to provide common functionally that is used at many places in design
  • A module provides the neccassary functionality to the higher level blocks through its port interface (Inputs and Outputs) but hides the internet implementation
  • That allows the modifies to modify internal without affecting the rest of the design

Example:

module OR (Y,A,B);

endmodule

  • Everything we write in Verilog must be in module, exception compiler directives
  • Declared with keyword "module", ended with "endmodule"
  • Each module must have
    • "module_name"
    • "module_terminal_list"(I/O)
  • Modules cannot be nested
