VERILOG

Data types

Net types

  • ✔️ Physical connection between structural element.
  • ✔️ Carries the value of the Signal it is connected to and transmit to the circuit block connected to it.
  • ✔️ If driving end of a net is floating, then net goes z.
  • Ex: wire

Net types

  • ✔️ Represent an abstract storage element.
  • ✔️ A net or wire connected to reg takes on the value stored in the reg and can be used as input to other circuit element.
  • Ex: reg
  • Default Values
    • net list : z
    • register : x

Scalar & Vectors

  • Entities represent single bit (wheter the bit is stored, changed or transferred) called scalar.

    • Ex: wire w1; //A 1- bit wire

  • Multiple lines carry signals in cluster are called scalar.

    • Ex1: wire [msb:lsb] wire1, wire2,....;

    Ex2: reg [msb:lsb] reg1, reg2,.....;

    Ex3: wire [6:0] Clear; // A 7-Bit wire;

  • reg [0:3] mem [0:63];

    • --An array of 64 4-Bit registers.

  • reg mem [0:4];

    • --An array of 64 1-Bit registers.

Ports

  • Ports provide the interface by which a module can communicate
  • Port Declaration:
    • ▶️input: input port
    • ▶️output: output port
    • ▶️inout: bidirectional port

    Input Declaration

      Scalar

      • input is the list of input identifiers
      • Ex: input A,B;

      Vector

      • input [range]
      • Ex: input[15:0] A,B;

    Output Declaration

    • Scalar Example: output c_out;
    • Vector Example: output[7:0] ACC, REG_IN;
