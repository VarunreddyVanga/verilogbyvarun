VERILOG (Operators)

Arthimetic Operators </>:

  • * = Multiply.
  • / = Divide.
  • + = Add.
  • - = Subtract.
  • % = Modulus.

Ex1: (5+10 = 15), (10%3 = 1),(5-10 = -5), (10/(-5) = -2)

Logical Operators </>:

  • && → Logical AND
  • || → Logical OR
  • ! → Logical NOT

Operands evaluate one bit value: 0,1 and x.

Result is one bit value.

  • A=1; A&&B → 1&&0 → 0
  • B=0; A||B → 1||0 → 1
  • C=x; C||B → x||0 → x

Relational Operators </>:

  • a < b (a less than b)
  • a > b (a greater than b)
  • a <= b (a less than or equal to b)
  • a >= b (a greater than or equal to b)
  • 1 is true
  • 0 is false
  • The result is a scalar
  • x if any operands are unknown.

Equality Operators </>:

  • a === b → a is equal to b, including x and z (Case euality).
  • a !== b → a is not equal to b, including x and z (Case inequality).
  • a == b → a equal to b, result may be unknown (Logical equality).
  • a != b = → a not equal to b, result may be unknown (Logically equality).

-> Operands are compared bit by bit with zero filling if the operands do not have same length.

-> Result is 0(false) or 1(true).

Ex:

4'bx001 === 4'bx001 = 1;

4'bx0x1 === 4'bx001 = 0;

4'bz0x1 === 4'bz0x1 = 1;

4'bx0x1 === 4'bx001 = 1;

5 == 10 = 0;

5 == 5 = 1;

Shift & Conditional Operators </>:

  • >> → Shift Right.
  • << → Shift Left.

    • a = 4,b1010; d = a>>2; // d = 0010 = a<<1 // c = 0100

  • - = Subtract.
  • % = Modulus.

Ex1: (5+10 = 15), (10%3 = 1),(5-10 = -5), (10/(-5) = -2)
